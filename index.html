Fractal One Software Looper

A minimalist software made for a simple and powerful looping experience in music.

FRACTAL Looper was made as the software solution of a music production looping dedicated device that needed an easy to the eye interface and also easy to use by the user without losing concentration from the music making process. I designed the user interface to met all the client requeriments and also collaborated in the front and back-end programming to give the right solutions and behaviours to the application. Now the software has been optimized and now is being tested to be released in windows, android in ios in december 2018.
